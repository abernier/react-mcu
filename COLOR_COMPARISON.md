## Material Theme Builder Screenshot

To compare with Material Theme Builder:

1. Visit: https://material-foundation.github.io/material-theme-builder/
2. Enter the source color: `#769CDF`
3. Select scheme: "Tonal Spot" (this is the default)
4. Keep contrast at standard (0)
5. Keep "Color match - Stay true to my color inputs" **unchecked** (this corresponds to `colorMatch: false`)
6. Take a screenshot of the color palette

**Instructions for taking the Material Theme Builder screenshot:**

- Navigate to the URL above
- Click on "Primary" or the color picker

## Expected Comparison

Both implementations should produce **identical** or **very similar** colors because:

1. Both use the same Material Design 3 color system
2. The react-mcu library uses the official `@material/material-color-utilities` package
3. The Material Theme Builder is the official reference implementation from Google

### Key Color Roles to Compare

#### Primary Colors

- **Primary**: Main brand color
- **On Primary**: Text/icons on primary
- **Primary Container**: Prominent container color
- **On Primary Container**: Text/icons on primary container
- **Primary Fixed**: Fixed primary for dark/light themes
- **Primary Fixed Dim**: Dimmed version of fixed primary
- **On Primary Fixed**: Text on fixed primary
- **On Primary Fixed Variant**: Alternative text on fixed primary

#### Secondary Colors

- Similar structure to Primary colors

#### Tertiary Colors

- Similar structure to Primary colors

#### Error Colors

- **Error**: Error state color
- **On Error**: Text/icons on error
- **Error Container**: Error container color
- **On Error Container**: Text/icons on error container

#### Surface Colors

- **Surface Dim**: Dimmed surface
- **Surface**: Default surface
- **Surface Bright**: Bright surface
- **Surface Container Lowest/Low/Default/High/Highest**: Container hierarchy
- **On Surface**: Text/icons on surface
- **On Surface Variant**: Variant text on surface
- **Outline**: Border/divider color
- **Outline Variant**: Alternative outline color

#### Additional Colors

- **Inverse Surface**: Inverse of surface
- **Inverse On Surface**: Text on inverse surface
- **Inverse Primary**: Inverse of primary
- **Scrim**: Overlay scrim
- **Shadow**: Shadow color

## Color Verification Code

You can also programmatically verify the colors by checking the generated CSS variables in the browser console:

```javascript
// In Storybook, open the browser console and run:
const root = document.documentElement;
const style = getComputedStyle(root);

console.log("Primary:", style.getPropertyValue("--mcu-primary").trim());
console.log("Secondary:", style.getPropertyValue("--mcu-secondary").trim());
console.log("Tertiary:", style.getPropertyValue("--mcu-tertiary").trim());
console.log("Error:", style.getPropertyValue("--mcu-error").trim());
// ... and so on for all color roles
```

## Notes

- The `react-mcu` library implements the Material Design 3 color system using the official utilities
- Color generation is deterministic - same input should always produce same output
- Any differences would indicate either:
  - A bug in the implementation
  - A difference in library versions
  - A configuration mismatch
