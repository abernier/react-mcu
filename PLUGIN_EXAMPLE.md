# Tailwind CSS v4 Plugin Example

This example shows how to use the react-mcu Tailwind plugin to generate color variables.

## Usage

```typescript
import { generateMcuTheme } from "react-mcu/tailwind.plugin";

// Generate theme with custom colors
const theme = generateMcuTheme({
  customColors: ["myCustomColor1", "myCustomColor2", "myCustomColor3"],
});

console.log(theme);
```

## Generated Output

The plugin generates:

1. **Base colors** - All Material Design 3 color roles:
   - `--color-background`, `--color-primary`, `--color-onPrimary`, etc.

2. **Color shades** - For primary, secondary, tertiary, error, neutral, and neutral-variant:
   - `--color-primary-50` through `--color-primary-950`
   - `--color-secondary-50` through `--color-secondary-950`
   - etc.

3. **Custom colors** - Based on your configuration:
   - `--color-myCustomColor1`, `--color-on-myCustomColor1`, etc.
   - Includes shades: `--color-myCustomColor1-50` through `--color-myCustomColor1-950`

## In Your CSS

After generating the theme, you can use it in your Tailwind CSS:

```css
@import "tailwindcss";

/* Theme generated by the plugin */
@theme inline {
  --color-background: var(--mcu-background);
  --color-primary: var(--mcu-primary);
  /* ... etc */
}
```

## Using with Tailwind Classes

Once the theme is set up, you can use Tailwind classes:

```tsx
<div className="bg-primary text-on-primary">Primary</div>
<div className="bg-myCustomColor1 text-on-myCustomColor1">Custom Color</div>
<div className="bg-primary-500">Primary shade 500</div>
```
